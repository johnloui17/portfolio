## [0.1.0] — 2026-03-11

### Added
- Next.js 16 portfolio project scaffold with App Router
- Minimal dark monochrome design system (`globals.css`)
- Framer Motion interactive spotlight — cursor-tracking purple/blue orb background
- Framer Motion scroll-triggered fade-up animations on all sections
- **Hero Section** — headline, sub-description, CTA buttons
- **Engineering Arsenal** — bento-box grid with Frontend, Backend, Security, Analytics, and Efficiency & Scale tiles
- **Selected Works** — Carland360, Marketfeed Systems, Financial Calculators & Web Stories
- Mobile responsive layout with 16px side padding breakpoint
- GitHub / LinkedIn navigation icons
- Jest unit tests for all page sections  
- Cypress E2E tests — navigation, content integrity, responsive viewports
- GitHub Actions CI pipeline (lint → build → test)
- GitHub Actions release pipeline (auto version bump + git tag on merge to master)
- PR template for consistent contribution workflow
